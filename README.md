**GrizzCal — Group Scheduling Web Application**

GrizzCal is a web-based scheduling tool designed to help student groups compare availability 
and coordinate shared events. Multiple users can view, create, and update events on the same calendar, making it easier to find overlapping free time and plan meetings efficiently.

This project focuses on usability, collaboration, and clear visual presentation of time-based data.  
Built as part of a web development coursework project.

---

**Features**

- Shared **multi-user calendar**
- Create, edit, and delete events
- Day and week schedule views
- Visual layout designed for class + work schedule comparison
- Lightweight interface, easy to modify and expand

---

**Tech Stack**

| Component | Technology |
|----------|------------|
| Front-End | HTML, CSS, JavaScript |
| UI Layout | CSS Grid / Flexbox |
| Data Model (Current Implementation) | Front-end local data storage (Prototype) |
| Planned Data Handling | Server + Shared Database (Future Implementation) |
